# Sudoku-Grid-Maker

Generates read-to-print PDF sudoku grids for this toy:
https://www.vertbaudet.fr/sudoku-des-couleurs-en-bois-fsc-multicolore.htm?ProductId=620031049&FiltreCouleur=6787

## Requirements
Python 3.X

## Install
```sh
./install.sh
```

## Usage
```sh
# See command line help
python sudoku.py -h
```
